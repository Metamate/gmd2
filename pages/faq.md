# FAQ

## General Information

??? question "Is there an FAQ for this course?"
    Yes, you are looking at it. This FAQ is designed to answer some of the most common questions about the course. If you have a question that is not answered here, feel free to contact the course coordinator, [Jakob Trigger Knop](mailto:jknr@via.dk).

??? question "What is the course about?"
    Making games from scratch and learning game design and architecture in the process.

??? question "What are the prerequisites for the course?"
    A basic understanding of Object-Oriented Programming (OOP) and C#.

??? question "What are the requirements for the course project?"
    Please see the [Project](./project.md) section for more information.

??? question "What is the exam like?"
    The exam is an oral exam where you draw a question from a pool of questions and have to answer it. For more information, see the [Exam](./exam.md) section.
